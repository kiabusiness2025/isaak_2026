# @isaak/brand

**Función:** colores, tipografías, assets, logos, fondos retro — la fuente de marca canónica.

`src/{colors,typography,tokens,motion,assets}.ts` expone los tokens tipados que consume
`apps/web`. `assets/` guarda los archivos fuente (logos, iconos, poses del personaje,
fondos, referencia) organizados por categoría — ver `assets/INVENTORY.md` para el estado
(`approved`/`candidate`) y uso de cada uno.
