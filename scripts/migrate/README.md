# scripts/migrate

Scripts de migración de datos/módulos desde el repo `isaak` (legacy) hacia este repo:
por ejemplo, exportar el schema de Prisma, migrar registros de `SeConnection`, o portar
contenido de `packages/integrations`.

Carpeta creada como parte del esqueleto de estructura — todavía sin scripts. Cada script que
se añada aquí debe corresponder a una ficha de migración concreta en `docs/migration/`
(origen, destino, riesgo).
